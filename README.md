FERIADO FELIZ COM PADRÕES :)

Por: Lívia Costa da Silva


Sistema em Java inspirado no aplicativo Letterboxd para avaliação de álbuns musicais.


O usuário pode avaliar álbuns, receber notificações e exportar dados utilizando diversos padrões de projeto estudados na disciplina.


Os padrões usados foram do capitulo 6 do livro Engenharia de Software Moderna do Marco Tulio Valente
Padrões utilizados: Fábrica, Singleton, Proxy, Adaptador, Fachada, Decorador, Strategy, Observador, Template Method, Visitor.

Referência:
Marco Tulio Valente. Engenharia de Software Moderna: Princípios e Práticas para Desenvolvimento de Software com Produtividade, Editora: Independente, 2020. (bibtex)


**** Padrões de Projeto Utilizados ****
- Factory -> criação de álbuns
- Singleton -> logger do sistema
- Proxy -> controle de acesso premium
- Adapter -> integração com o spotify
- Facade -> simplificação das operações principais
- Decorator -> avaliação completa com texto + nota
- Strategy -> estratégias diferentes de avaliação
- Observer -> notificação de seguidores
- Template Method -> fluxo padrão de avaliação
- Visitor -> exportação de dados
  

**** Funcionalidades ****
- Criar usuários
- Avaliar álbuns
- Notificar seguidores
- Exportar avaliações
- Integração simulada com o spotify
- Controle de acesso premium
